## What I'm building

  algo trading workbench. ArcticDB bars store, vectorbt research harness with
  IS/OOS walk-forward gating, prop-firm-style risk engine, scheduler daemon.
  Includes 8 role-specific Claude Code agents, opus/sonnet/haiku tiered.
- **[unified-search-mcp](https://github.com/Burkswill2/unified-search-mcp)** — MCP
  server unifying semantic search across papers, experiments, hypotheses, and
  notes. FastMCP + ChromaDB + SQLite FTS5.
- **[markowitz-api](https://github.com/Burkswill2/markowitz-api)** — FastAPI
  service for mean-variance portfolio optimization via SciPy SLSQP.
- **[finbert-sentiment-trader](https://github.com/Burkswill2/finbert-sentiment-trader)** —
  2024 sentiment-trading experiment, archived as a post-mortem on position
  sizing and signal-quality failures.
